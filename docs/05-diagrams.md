## Offering Lifecycle

### State Machine

![Offering State Machine](./images/state_machine.png)

### Offering Statuses Explained

| Status | What It Means |
|--------|---------------|
| **SCHEDULED** | Offering created, waiting for start date |
| **PENDING** | Start date passed, but not yet activated |
| **ACTIVE** | Open for investments |
| **PAUSED** | Temporarily stopped (can be resumed) |
| **SUSPENDED** | Stopped by admin (requires unsuspend) |
| **ACTIVE_SOFTCAP_MET** | Minimum funding goal reached, still accepting investments |
| **COMPLETED** | End date reached or all tokens sold |
| **SUCCESSFUL** | Offering ended successfully |
| **FAILED_SOFTCAP** | End date reached but minimum goal not met |
| **REFUNDED** | Investors refunded after failed soft cap |
| **CANCELED** | Offering canceled before completion |
| **TERMINATED** | Offering ended early by admin |
| **ARCHIVED** | Final state, offering closed |
